# Parts Styles

This is a library of css styles for parts in the Konstruct ecosystem. Can be used stand-alone, or as part of the react parts.

- [Buttons](#buttons)

## Buttons

### Installation

**Installing the library:** ```yarn add @traaidmark/konstruct-parts-css;```

* **With js:** ```import '@traaidmark/konstruct-parts-css/buttons';```
* **With HTML:** ```<link rel="stylesheet" type="text/css" href="" />```
* **With css:** ```@import ...```

### Usage

The ```.part-button``` can be applied equally to ```<a>``` & ```<button>``` elements, and of course will work with most JS link libraries too.

<Source
  code={dedent`
    <a href="#" className="part-button">Button</a>
    <button className="part-button">Button</button>
  `}
/>

### Variables

This library provides a set of variables to help you work consistently with buttons. You can override these at root level or use them to style your own buttons.

<Source
  code={dedent`
    --button-radius: var(--radius-small);
    --button-border-width: 2px;
    --button-text-color: var(--color-blue);
    --button-text-color-hover: var(--color-darker);
    --button-bg-color: var(--color-darker);
    --button-bg-color-hover: var(--color-blue);
    --button-border-color: var(--color-darker);
    --button-border-color-hover: var(--color-blue);
  `}
/>

### Theming

Since buttons are styled with variables, you can easily create your own button styles like this example:

<Source
  code={dedent`
    .part-button-danger {
      --button-text-color: var(--color-darker);
      --button-text-color-hover: var(--color-red);
      --button-bg-color: var(--color-red);
      --button-bg-color-hover: var(--color-darker);
      --button-border-color: var(--color-red);
      --button-border-color-hover: var(--color-darker);
    }
  `}
/>

**Note:** It's a good idea to stick to the button's naming convention (```.part-button-[style-name]```) to avoid any hassles with css specificity down the line.
