## Installation

Using Yarn:
<Source
  code={dedent`
    yarn add @traaidmark/konstruct-part-loader
  `}
/>

Using NPM:
<Source
  code={dedent`
    npm install @traaidmark/konstruct-part-loader
  `}
/>
