# Sticker

The `<Sticker />` component is a small extension on the data display component.

- [Installation](#installation)
- [Dependencies](#dependencies)
- [Usage](#usage)
- [Props](#props)

## Installation

<Source
  code={dedent`
    yarn add @traaidmark/konstruct-part-sticker;
  `}
/>

## Style Dependencies

Konstruct components separate js from css, which makes the css libraries platform agnostic.

So you need to install the Cement library too, and import the required files.

<Source
  code={dedent`
    yarn add @traaidmark/konstruct-cement;
  `}
/>

Import the following css libraries wherever you choose to import your css files. I suggest a single location for ease of maintenance.

* `@traaidmark/konstruct-cement/data.css`
* `@traaidmark/konstruct-cement/sticker.css`

## Usage

<Source
  code={dedent`
    import React from 'react';
    import Sticker from '@traaidmark/konstruct-parts-sticker';
    
    const DemoComponent = () => {
      
      return (
        <Sticker
          label="Amount"
          prefix="usd"
          value="23.00"
        />
      );
    };
    export default DemoComponent;
  `}
/>
