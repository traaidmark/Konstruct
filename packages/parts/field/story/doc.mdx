# Field

This is a fairly simple container part to wrap your input fields within.

- [Installation](#installation)
- [CSS](#css)
- [React Props](#react-props)

## Installation

Using Yarn:
<Source
  code={dedent`
    yarn add @traaidmark/konstruct-part-field
  `}
/>

Using NPM:
<Source
  code={dedent`
    npm install @traaidmark/konstruct-part-field
  `}
/>
