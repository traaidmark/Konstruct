# Field

Super small component to display a small flag with themeable styles.

The component comes with default styling and a primary / secondary variant that is updated based on color settings, but you can easily create your own variants with simple css variables.

## Installation

<Source
  code={dedent`
    yarn add @traaidmark/konstruct-part-flag;
  `}
/>

## Dependencies

This component needs `@traaidmark/konstruct-foundation` to work if you want to use default styles.

## Usage

<Source
  code={dedent`
    import React from 'react';
    import Flag from '@traaidmark/konstruct-parts-flag';
    
    const DemoComponent = () => {
      return (
        <Flag
          variant="info"
          text="Info"
        />
      );
    };
    export default DemoComponent;
  `}
/>

## Theming

Variants are created by creating a set of css variables, according to a set structure like the below example:

<Source
  language='css'
  code={dedent`
    :root {
      --flag-[NAME]--text-color:        var(--color-darker);
      --btn-[NAME]--background-color:   var(--color-blue);
    }
  `}
/>

The variant name is taken from the css variable name itself: `--flag-[NAME]--text-color`.

You can then use your new variant by entering `[NAME]` as your variant in the component:

<Source
  language='html'
  code={dedent`
    <flag variant="[NAME]" text="Info" />
  `}
/>
