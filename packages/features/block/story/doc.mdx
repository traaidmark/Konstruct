## Installation

Using Yarn:
<Source
  code={dedent`
    yarn add @traaidmark/konstruct-feature-block
  `}
/>

Using NPM:
<Source
  code={dedent`
    npm install @traaidmark/konstruct-feature-block
  `}
/>

## CSS

### Classes

| Class              | Description                                   |
| ------------------ |:---------------------------------------------:|

| Variable             | Default Values         | Description           |

| --block-border-color | var(--color-light)

Visit our theming guide if you want to learn how creating your own variants work.
