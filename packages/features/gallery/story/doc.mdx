# Gallery

The simplest implementation of an image gallery.

- [Installation](#installation)
- [CSS](#css)
- [React Props](#react-props)

## Installation

Using Yarn:
<Source
  code={dedent`
    yarn add @traaidmark/konstruct-feature-gallery
  `}
/>

Using NPM:
<Source
  code={dedent`
    npm install @traaidmark/konstruct-feature-gallery
  `}
/>
