# Konstruct Cement - Parts

Cement Parts is a css library to add common interface bits to your project.

## Installation

<Source
  code={dedent`
    yarn add @traaidmark/konstruct-cement-parts;
  `}
/>

## Parts

* Button: `import '@traaidmark/konstruct-cement-parts/button.css';`
* Flag: `import '@traaidmark/konstruct-cement-parts/flag.css';`
* Data: `import '@traaidmark/konstruct-cement-parts/data.css';`
* Sticker: `import '@traaidmark/konstruct-cement-parts/sticker.css';`
