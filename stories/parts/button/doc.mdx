# Button

The `<Button />` component is a smart component that can render out a standard button or standard link based on the context it's used in.

The component comes with default styling and a primary / secondary variant that is updated based on color settings, but you can easily create your own variants with simple css variables.

## Usage

<Source
  code={dedent`
    import React from 'react';
    import Button from '@traaidmark/konstruct-parts-button';
    
    const DemoComponent = () => {
      const btnAction = () => console.log('Click Happened!');
      return (
        <Button
          variant="primary"
          size="default"
          onClick={ () => btnAction() }
        >
          Button Label
        </Button>
      );
    };
    export default DemoComponent;
  `}
/>

## Theming

Variants are created by creating a set of css variables, according to a set structure like the below example:

<Source
  language='css'
  code={dedent`
    :root {
      --btn-[NAME]--color:   var(--color-secondary);
      --btn-[NAME]--colorH:  var(--color-accent);
      --btn-[NAME]--bg:      var(--color-accent);
      --btn-[NAME]--bgH:     var(--color-secondary);
      --btn-[NAME]--border:  var(--color-accent);
      --btn-[NAME]--borderH: var(--color-secondary);
    }
  `}
/>

The variant name is taken from the css variable name itself: `--btn-[NAME]--color`.

If you prepend `[NAME]` with `icon-` (ie: `--btn-icon-[NAME]--color`), the button will become an icon button.

You can then use your new variant by entering `[NAME]` as your variant in the component:

<Source
  language='html'
  code={dedent`
    <Button variant="[NAME]">Sign Up</Button>
  `}
/>
