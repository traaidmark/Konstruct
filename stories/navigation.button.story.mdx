## Variants

The `variant` name is derived from a matching set of css variables, as you can see in the below set of examples:

<Source
  language='html'
  code={dedent`
    <Button variant="primary">Sign Up</Button>
  `}
/>

The variant name is taken out of the css variable set below:

<Source
  language='css'
  code={dedent`
    :root {
      --btn-primary--color:   var(--color-secondary);
      --btn-primary--colorH:  var(--color-accent);
      --btn-primary--bg:      var(--color-accent);
      --btn-primary--bgH:     var(--color-secondary);
      --btn-primary--border:  var(--color-accent);
      --btn-primary--borderH: var(--color-secondary);
    }
  `}
/>

To create your own button styles, you simply need to add a set of variables like above, and give it a unique name.
